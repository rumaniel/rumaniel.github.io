## Overview
다크모드 시스템과 타이포그래피 개선을 추가합니다.

## Key Features

### Dark Mode System
- OS/브라우저 테마 자동 감지 (`prefers-color-scheme`)
- 수동 토글 버튼 (헤더에 moon/sun 아이콘)
- localStorage 기반 사용자 설정 영구 저장
- FOUC(Flash of Unstyled Content) 방지를 위한 inline bootstrap script

### Typography & Styling
- **한글 가독성**: Noto Sans KR, Noto Serif KR 폰트 추가
- **헤딩 계층**: H1-H6 명확한 크기/두께 차별화 (`clamp()` 반응형)
- **다크모드 대비**: 코드 블록, 테이블, 카드 요소 명도 조정
- **토큰 기반 시스템**: CSS custom properties로 일관성 유지

### Code Quality
- 인라인 스타일 제거 (categories.md)
- 컴포넌트 클래스 통합 (home/categories/archives card 공통화)
- CSS 중복 제거 (~35줄 감소)

## Changes
