# Jekyll 플러그인 도입 계획

## 플러그인 우선순위 목록

### 우선순위 1 (필수/권장)

#### jekyll-sitemap
- **용도**: XML sitemap 자동 생성 (SEO)
- **현 상태**: 미도입
- **GitHub Pages 호환성**: 화이트리스트 포함 (안정)
- **이유**: 검색 엔진 크롤링 최적화
- **도입 난이도**: ⭐ (낮음)
- **상태**: `TODO` → 우선 도입

#### jekyll-archives
- **용도**: 날짜/카테고리별 아카이브 페이지 자동 생성
- **현 상태**: 미도입
- **GitHub Pages 호환성**: 화이트리스트 미포함 → Actions에서만 사용 가능
- **이유**: 포스트 분류/탐색 개선
- **도입 난이도**: ⭐ (낮음, 단 배포 파이프라인 필요)
- **상태**: `TODO` → Epic 4-2 도입 예정

### 우선순위 2 (선택/개선)

#### jekyll-toc (Table of Contents)
- **용도**: 포스트 목차 자동 생성
- **현 상태**: 미도입
- **GitHub Pages 호환성**: 화이트리스트 미포함
- **이유**: 긴 글 가독성 개선
- **도입 난이도**: ⭐⭐ (중간, Liquid 필터 통합)
- **상태**: `BACKLOG` (이후)

#### jekyll-paginate-v2
- **용도**: 포스트 목록 페이징 (현재 전체 로드)
- **현 상태**: 미도입
- **GitHub Pages 호환성**: 화이트리스트 미포함
- **이유**: 홈 페이지 성능 개선 (포스트 수 증가 시)
- **도입 난이도**: ⭐⭐ (중간)
- **상태**: `BACKLOG` (향후)

#### jekyll-related-posts
- **용도**: 관련 포스트 추천 (포스트 하단)
- **현 상태**: 미도입
- **GitHub Pages 호환성**: 화이트리스트 미포함
- **이유**: 사용자 체류/탐색 개선
- **도입 난이도**: ⭐⭐ (중간)
- **상태**: `BACKLOG` (향후)

### 우선순위 3 (보류)

#### jekyll-time-to-read
- **용도**: 읽기 시간 자동 계산 (현재 include로 수동)
- **현 상태**: 수동 구현 중 (`_includes/reading-time.html`)
- **GitHub Pages 호환성**: 화이트리스트 미포함
- **이유**: 현재 수동 구현 충분
- **상태**: `HOLD` (현 수동 구현 유지)

#### jekyll-compose
- **용처**: 포스트 스켈레톤 생성 보조
- **현 상태**: 미도입
- **이유**: 로컬 개발자 생산성 도구 (배포 영향 없음)
- **상태**: `BACKLOG` (선택)

## 도입 기준 (체크리스트)

도입 전 반드시 검토:

- [ ] **호환성**: 현재 Jekyll 4.x, minima 2.5.x와 충돌 없는가?
- [ ] **GitHub Pages 제약**: Actions 빌드 환경에서 정상 작동하는가?
  - 화이트리스트 포함 → 어디서든 가능
  - 화이트리스트 미포함 → Actions 빌드에서만 사용 가능
- [ ] **보안**: gem 발행처/업데이트 활동 확인
- [ ] **유지보수성**: 의존성 추가 수 최소화
- [ ] **성능**: 빌드 시간 증가량 테스트
- [ ] **문서화**: `Gemfile`, `_config.yml` 주석 추가

## 1차 도입 예정 (Epic 4-1)

### jekyll-sitemap

**Gemfile에 추가**
```ruby
group :jekyll_plugins do
  gem "jekyll-sitemap"
end
```

**_config.yml에 추가**
```yml
plugins:
  - jekyll-sitemap
```

**빌드 후 확인**
- `_site/sitemap.xml` 생성 여부
- 모든 포스트/페이지 포함 확인
